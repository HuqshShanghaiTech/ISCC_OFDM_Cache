# Integrated Sensing-Communication-Computation Based Online Federated Learning With Limited Cache in OFDM Systems

This repository contains the official implementation of our paper "Integrated Sensing-Communication-Computation Based Online Federated Learning With Limited Cache in OFDM Systems".


## How to use it

Following are the brief introductions of each file.

* main_fed.py is the main file.
* data_set.py is the pre-processor of the dataset.

Notice that for the optimization algorithm about the network resources, this version just provides an example value.
